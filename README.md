# Sign Language Kiosk for the Deaf 


## 📖 Abstract

Deaf, dumb, and hard-of-hearing (HoH) individuals face communication barriers in public service environments due to limited awareness of Indian Sign Language (ISL) and dependence on interpreters. Existing solutions often lack real-world applicability or rely on full automation, reducing natural and inclusive interaction.

This project presents an **AI-powered sign language communication kiosk** enabling **bi-directional translation** between ISL and spoken language using:
- 📷 Computer Vision
- 🎙️ Speech Processing
- 🧑‍💻 3D Animated Avatar

Implemented across two networked devices, the system functions as an assistive intermediary, providing a practical, real-time, and accessible communication solution for public environments.

## 🚀 Introduction

Effective communication in public environments such as hospitals, banks, and government offices remains challenging for deaf and hard-of-hearing individuals due to limited awareness of Indian Sign Language (ISL) and the shortage of interpreters. 

Research has shown that image processing and neural network-based systems can translate sign language into text or speech (Dhake et al., 2020), and recent deep learning approaches using Convolutional Neural Networks (CNNs) have improved the accuracy of ISL recognition (Dikshant et al., 2024). Modern AI systems now integrate computer vision, speech recognition, and natural language processing to enable real-time sign-to-text and speech-to-sign translation (Patil et al., 2025). Multimodal kiosk-based systems using text, speech, and animated avatars have also proven effective in enhancing accessibility (Aran et al., 2008).

Based on these advancements, this project proposes an AI-powered sign language communication kiosk for Indian citizens that supports **bi-directional communication**. 

- It **converts ISL gestures into text and speech** using deep learning models.
- It **translates spoken responses into ISL** through an animated avatar. 

By combining computer vision, speech processing, and avatar-based sign rendering, the system aims to provide an inclusive, scalable, and independent communication solution for public service environments.
